# Merchant Reputation System

## Goal

Merchant reputation makes trust visible without reducing a merchant to a single
star rating.

## Foundation Factors

- Customer reviews
- Repeat customers
- Response rate
- Community activity
- Founder status
- Official Store status
- Longevity
- Live commerce participation

Sprint 17 uses existing merchant signals to calculate a display-only trust
score.

## Tiers

| Tier               | Foundation threshold |
| ------------------ | -------------------- |
| Bronze Merchant    | 0                    |
| Silver Merchant    | 50                   |
| Gold Merchant      | 70                   |
| Master Merchant    | 85                   |
| Legendary Merchant | 95                   |

## Rules

- Reputation must not override evidence or consumer protections.
- Founder status contributes context, not immunity.
- Paid placement must never secretly increase trust.
- Reviews require future anti-abuse and moderation systems.
- Merchants must be able to understand reputation factors.
- High-impact enforcement remains human governed.
