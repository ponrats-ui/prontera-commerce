# Prontera Art Bible

## Core Art Direction

Name:

```text
Prontera Commerce Original Style
```

Prontera Commerce must be an original fantasy commerce world. It may be inspired by the general ideas of warm fantasy towns, merchant districts, community markets, trade routes, and game-like commerce, but all characters, cities, maps, NPCs, creatures, UI, names, and visual identity must be original.

Core principles:

- Original fantasy commerce world
- Warm merchant civilization
- Friendly but not childish
- Game-like but business-ready
- Trustworthy for real merchants
- Family-friendly
- Global-ready
- Avoid copying any known franchise

## Visual Keywords

Use:

- warm
- inviting
- handcrafted
- merchant guild
- market town
- trade routes
- sky gates
- artisan shops
- cozy technology
- magical commerce
- human-centered

Avoid:

- Ragnarok-like
- Pixar-like
- Disney-like
- Ghibli-like
- Animal Crossing-like
- Minecraft-like
- Shopee-like
- Lazada-like

## Character Design Rules

Characters should be:

- original
- merchant-focused
- friendly
- readable at small sizes
- customizable
- culturally adaptable
- non-infringing

Do not use:

- Ragnarok classes
- Ragnarok monsters
- known character silhouettes
- copyrighted costumes
- copied NPC designs
- cloned mascot shapes

## NPC Design Rules

NPCs should be:

- shopkeepers
- artisans
- guides
- porters
- guild clerks
- market announcers
- delivery helpers
- AI merchant assistants

NPCs must:

- disclose AI identity when AI-powered
- not impersonate real people without consent
- not copy fictional characters

## Creature and Mascot Rules

Create an original creature taxonomy.

Examples:

- Glow Sprites
- Parcel Pets
- Coinlings
- Gate Wisps
- Cart Buddies
- Lantern Foxes

Rules:

- no Poring-like creatures
- no slime clones
- no monster designs from existing games
- no near-copy silhouettes

## City and Map Design Rules

Cities should be based on commerce functions, not copied fantasy cities.

Examples:

- Central Merchant City
- Harbor Market
- Artisan Valley
- Sky Gate District
- Spice Quarter
- Textile Row
- Tech Bazaar
- Greenhouse Market

Avoid:

- copied city names
- copied map layouts
- copied landmark silhouettes
- copied town square designs

## Naming Rules

Names must be original.

Avoid names from:

- Ragnarok Online
- Disney
- Pixar
- Nintendo
- Square Enix
- Studio Ghibli
- famous fantasy franchises

Use original naming patterns:

- Aether Market
- Velora Port
- Lumina Gate
- Novara Bazaar
- Solmere District
- Brighthaven Row

## UI and UX Style

UI should feel:

- merchant OS
- fantasy commerce
- clean
- trustworthy
- professional
- warm

Avoid:

- copying MMO UI
- copying marketplace UI
- copying game inventory grids from known titles

## Brand Differentiation Statement

Prontera Commerce is:

```text
Merchant OS + Marketplace + Live Commerce + Commerce Gate Network + Future Open World Commerce.
```

Prontera Commerce is not:

- a Ragnarok clone
- a game clone
- a Disney/Pixar-style character platform
- a metaverse skin of an existing marketplace

## Production Rule

All public-facing visual work must preserve the Prontera Commerce Original Style and pass the [IP Safety Guidelines](./ip-safety-guidelines.md) before production use.
