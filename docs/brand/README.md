# Brand Documentation

This directory preserves official brand direction, art rules, and intellectual property safety guidance for Prontera Commerce.

## Prontera Art Bible

[Prontera Art Bible](./prontera-art-bible.md) defines the Prontera Commerce Original Style: an original fantasy commerce world that feels warm, merchant-focused, trustworthy, family-friendly, global-ready, and business-ready.

## IP Safety Guidelines

[IP Safety Guidelines](./ip-safety-guidelines.md) define what Prontera Commerce must avoid when creating characters, NPCs, mascots, cities, maps, UI, names, prompts, and public-facing assets.

## Operating Rule

Prontera Commerce may draw from broad ideas such as warm market towns, merchant guilds, trade routes, artisan shops, and magical commerce, but it must not copy any existing franchise, studio style, marketplace brand, character silhouette, mascot, map, lore, UI, logo, music, or distinctive visual identity.

All public-facing art direction must be reviewed by COM before production use.
