# v0.3.0 Promotion & Pricing

Date: 2026-06-16

Status: Internal Alpha

## Summary

Sprint 8 introduces the Promotion & Pricing Engine Foundation for Prontera Commerce.

Merchants can create promotion campaigns, vouchers, and customer pricing tiers. Checkout evaluates automatic campaigns, voucher codes, and CRM customer group pricing before creating a pending order.

## Included

- Promotion campaign data model
- Promotion rule data model
- Voucher data model
- Customer pricing tier data model
- Promotion API
- Promotion evaluation endpoint
- Checkout voucher support
- Checkout customer group pricing support
- Order applied promotion snapshot
- Merchant dashboard promotions page
- Promotion engine tests

## Evaluation Strategy

Sprint 8 uses:

```text
BEST_DISCOUNT_WINS
```

Stacking multiple campaigns is intentionally deferred.

## Excluded

- Loyalty points
- Referral system
- Affiliate program
- AI pricing
- Payment gateways
- Marketing automation
